# ai-architect-global — AI 건축가 글로벌 플랫폼

AI 건축가 글로벌 랜딩 페이지 / 판매 플랫폼.
Lemon Squeezy 결제 연동.

## 빠른 시작

```bash
npm install
npm run dev       # 개발 서버
npm run build     # 프로덕션 빌드
npm start         # 프로덕션 서버
```

## 환경 변수 설정

`.env` 파일을 참조하여 필요한 환경 변수를 설정합니다.

## 기술 스택

- **Next.js 16**, TypeScript
- **Tailwind CSS**
- **Lemon Squeezy** (결제)
- **Vercel** 배포 (production/staging 브랜치)
